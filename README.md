# GCars

Programa realizado en Python para administrar un inventario de vehículos usados.

## Funciones

1. Agregar vehículo.
2. Buscar vehículo por patente.
3. Actualizar vehículo.
4. Ver listado de vehículos.
5. Salir.

## Archivos

- main.py → menú principal.
- utils.py → lectura y escritura del archivo JSON.
- vehiculos.json → almacenamiento de los vehículos.

## Ejecutar

```bash
python main.py
```
